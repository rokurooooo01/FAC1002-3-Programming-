# FAC1002-3-Programming
The entire collection of FAC1002/1003 Programming tutorials from Pusat Asasi Sains Universiti Malaya. Hopefully this would be helpful for upcoming batch who will be continuing here.

# what is this?
This is just a small intiative by me to help all upcoming PASUMians in this subject.

# how this works?
You'd run these codes using any C++ compilers or IDEs you'd like, or just look at how the codes are structured since that's pretty much the answers lol. For each '.cpp' file, it contains all the questions, arranged into a mini program. You'd be given options to navigate to any of the questions and try to 'test' the questions.

# note
This repo only contains the coding questions, not the theoretical questions.
